## Overview

Atlas is a modern intelligence platform built for the Normies ecosystem.

Unlike traditional NFT marketplaces or explorers, Atlas provides a real-time intelligence layer that allows users to explore, monitor and understand everything happening across the Normies ecosystem through live data.

Atlas transforms raw blockchain and metadata into an intuitive experience centered around discovery, ecosystem activity and insights.

---

## Why Atlas?

Every successful blockchain ecosystem eventually develops an intelligence layer.

Ethereum has Etherscan.

DeFi has DefiLlama.

Analytics has Dune.

Wallet Intelligence has Arkham.

Atlas brings this concept to the Normies ecosystem by providing a single platform for discovering collections, monitoring activity and exploring ecosystem data powered entirely by the official Normies API.

# Features

## Home

A live ecosystem overview showing the current state of the Normies ecosystem.

Features include:

- Live ecosystem metrics
- Trending Normies
- Recent discoveries
- Live activity preview
- Featured insights
- Universal search

---

## Explore

Atlas' core discovery experience.

Users can:

- Search Normies
- Filter by traits
- Browse collections
- Discover rare Normies
- View ownership
- Inspect metadata
- Explore history

Every result is fetched directly from the official API.

---

## Signals

A real-time activity feed for the entire ecosystem.

Monitor:

- Canvas updates
- Zombie conversions
- Transfers
- Legendary events
- Burn activity
- Ecosystem updates

---

## Watchlist

A personalized workspace where users can:

- Save Normies
- Watch wallets
- Save searches
- Receive ecosystem updates
- Track activity

---

## Universal Search

Search from anywhere inside the application.

Supports:

- Normie IDs
- Wallet addresses
- Traits
- Metadata
- Owners
- Zombie status
- Legendary status

# Authentication

Atlas uses modern Web3 authentication powered by Privy.

Users can sign in using:

- Wallet
- Google
- X (Twitter)

Authentication is displayed as a modal above the landing page rather than redirecting users to a separate login page.

# Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate into the project

```bash
cd atlas
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```

---

# Environment Variables

Create a `.env` file in the project root.

Example:

```env
VITE_NORMIES_API_URL=

VITE_NORMIES_API_KEY=

VITE_PRIVY_APP_ID=

VITE_PRIVY_CLIENT_ID=
```

> Replace the values above with your actual credentials.

# Data

Atlas does **not** use mock or placeholder data.

All ecosystem information displayed throughout the application is retrieved from the official Normies API.

This includes:

- Normies
- Traits
- Metadata
- Ownership
- Activity
- Search
- Signals
- Watchlist data

The application has been designed to gracefully handle loading, empty and error states while awaiting API responses.
